# Huebner Inverter - Web Interface

Web Interface for [Huebner Inverter Project](https://github.com/jsphuebner/stm32-sine) controlling 3-phase AC Induction Motors.

> **All-in-One**
> * RS232-USB Drivers
> * Inverter Parameters
> * Sensor Graphs
> * Firmware Flash
> * Wiring Diagrams
> * PCB Schematics
> * Component List
> * Encoder Builder
> * IGBT Test
> * CAN Setup
> * Auto Tunning
> * Source Code Compilers
